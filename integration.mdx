# Integration Guide

This guide walks you through authenticating with the Deets API, creating a secure checkout page, and customizing the payment experience.

> If you haven't set up your environment or obtained your API credentials, start with the [Getting Started](/getting-started) guide.

---

## 1. Authenticate with the Deets API

Obtain an authentication token using your API Key and App Key. This token is required for all subsequent API requests.

<Tabs>
  <Tab title="Node.js">

```js
const axios = require('axios');
const response = await axios.post(
  'https://test.digitszapi.com/test/auth/token',
  {
    data: {
      type: 'auth',
      attributes: {
        appKey: 'YOUR_APP_KEY'
      }
    }
  },
  {
    headers: {
      'x-api-key': 'YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  }
);
const token = response.data.data.attributes.appToken;
```
  </Tab>
  <Tab title="Python">

```python
import requests
url = "https://test.digitszapi.com/test/auth/token"
headers = {
    "x-api-key": "YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "data": {
        "type": "auth",
        "attributes": {
            "appKey": "YOUR_APP_KEY"
        }
    }
}
response = requests.post(url, json=data, headers=headers)
token = response.json()["data"]["attributes"]["appToken"]
```
  </Tab>
  <Tab title="cURL">

```bash
curl --location "https://test.digitszapi.com/test/auth/token" \
  --header "x-api-key: YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "data": {
      "type": "auth",
      "attributes": {
        "appKey": "YOUR_APP_KEY"
      }
    }
  }'
```
  </Tab>
</Tabs>

## 2. Create a Checkout Page

Use your authentication token to generate a secure, customizable checkout page for your users.

<Tabs>
  <Tab title="Node.js">

```js
const response = await axios.post(
  'https://test.digitszapi.com/test/generator/genPayment',
  {
    merchant_id: 'YOUR_MERCHANT_ID',
    is_capture_email: true,
    is_capture_zip: true,
    enable_token: true,
    payment_version: 'v4',
    style: {}
  },
  {
    headers: {
      'x-api-key': 'YOUR_API_KEY',
      'Authorization': 'Bearer YOUR_AUTH_TOKEN',
      'Content-Type': 'application/json'
    }
  }
);
const checkoutUrl = response.data.object.url;
```
  </Tab>
  <Tab title="Python">

```python
import requests
url = "https://test.digitszapi.com/test/generator/genPayment"
headers = {
    "x-api-key": "YOUR_API_KEY",
    "Authorization": "Bearer YOUR_AUTH_TOKEN",
    "Content-Type": "application/json"
}
data = {
    "merchant_id": "YOUR_MERCHANT_ID",
    "is_capture_email": True,
    "is_capture_zip": True,
    "enable_token": True,
    "payment_version": "v4",
    "style": {}
}
response = requests.post(url, json=data, headers=headers)
checkout_url = response.json()["object"]["url"]
```
  </Tab>


## 3. Customize the Checkout Experience

You can personalize the checkout page by adding URL query parameters to pre-fill customer details, adjust styling, and enable additional features.

| Parameter         | Type    | Required | Description                        |
|-------------------|---------|----------|------------------------------------|
| amount            | Integer | Yes      | Amount in cents (e.g., 1000 for $10.00) |
| email             | String  | No       | Customer's email address           |
| zip               | String  | No       | Customer's ZIP code                |
| invoice           | String  | No       | Unique invoice number              |
| splitConfig       | JSON    | No       | Split payment configuration        |

**Example:**
```
https://checkout.example.com?amount=1000&email=customer@example.com&zip=12345
```
