# API Reference

Welcome to the Payvia/Deets API Reference. Here you'll find detailed documentation for all available endpoints, authentication, and webhooks.

---

## How to Use This Reference

- Browse the available endpoints below for request/response details and usage examples.
- See the [Integration Guide](/integration) for authentication and setup instructions.
- Use the [Payments](/checkout) guide for end-to-end payment flows.

---

## Authentication

All API endpoints require authentication using a Bearer token and your API key.

- `Authorization: Bearer {token}`
- `x-api-key: {your-api-key}`

Obtain your token by following the [Integration Guide](/integration).

## Best Practices

- Always use HTTPS for all API requests.
- Store your API credentials securely and never expose them in client-side code.
- Handle errors gracefully and refer to the response codes for troubleshooting.
